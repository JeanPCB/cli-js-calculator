# icalc-js
Uma calculadora de terminal simples usando JavaScript e Node.js

## Sobre
Projeto para treinar lógica aritmética e interação dentro do terminal.

Uso do básico de Node.js:
- Controle de processo
- Manipulação de argumentos
- Variável de ambiente

Operações aritméticas em JavaScript:
- Soma
- Subtração
- Multiplicação
- Divisão

## Instalação
```bash
git clone https://github.com/JeanPCB/icalc-js.git
cd icalc-js
npm link
```

## Uso
```bash
calc add arg1 arg2
calc sub arg1 arg2
calc mult arg1 arg2
calc div arg1 arg2
```
**arg1/arg2 => operandos da expressão, eg.:**
```bash
calc add 2 2
```
Resultado: 4
